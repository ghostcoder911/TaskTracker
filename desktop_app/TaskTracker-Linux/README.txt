Task Tracker - Team Check-ins (Linux)
=======================================

Quick Start:
1. Open terminal in this folder
2. Run: ./TaskTracker
3. Enter your name
4. Answer the questions
5. Done! Data saved to Google Sheets

Requirements:
- Internet connection (for Google Sheets sync)
- Linux/Ubuntu

Files in this folder:
- TaskTracker (the app)
- credentials.json (Google credentials - keep safe!)
- config.json (spreadsheet configuration)
- README.txt (this file)

Troubleshooting:
- If "permission denied": run chmod +x TaskTracker
- If it doesn't start: make sure you have internet connection
- Data is backed up locally to checkins_backup.json

Contact your team admin for help.
